# 项目介绍
一，核心功能(模块与关键特性)
1，视觉底座(The Canvas)，基于WebGL/Shader的全屏动态背景。双模式：Rainy(雨天),Snowy(下雪).
2，沉浸编辑器(The Deck)。无边框毛玻璃输入框，三种内置字体。3，氛围控制台(Vibe Mixer)。隐藏式浮窗。滑杆控制：雨量，模糊度，白噪音音量，音乐融合度。
4，心流计时器(Zen Timer)。极简线性/环形消融倒计时。
5，数据管理(Privacy)。纯前端运行，LocalStorage 实时暂存，需要支持markdown渲染，公式渲染。然后当用户编辑失去焦点时，自动保存到本地
6. 支持双链笔记
7. 沉浸式模式退出可以进入文件夹
8. 支持自定义字体
9. 支持写后即焚的模式，也就是清空的时候，会显示文字随烟消散

二，视觉规范(Visual Vibe)
1。风格：赛博禅意(Cyber Zen)/极简主义。

2。交互：
-所有的UI 元素在非活跃状态下透明度趋近于0，鼠标移入时优雅淡入。
-模式切换器：位于屏幕顶部的极简文字(Rain/Star)，仅在鼠标悬浮在顶部50px区域时显现。
-统一逻辑：无论切换到哪个模式，毛玻璃文字框的状态，内容，字体设置均保持一致。
